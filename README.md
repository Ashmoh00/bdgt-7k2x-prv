# ميزانيتي — موقع شخصي على GitHub Pages

موقع كامل لمتابعة المصاريف من رسائل البنك السعودية، يحلل ويصنّف ويعرض لوحة بيانات.

## النشر على GitHub Pages (٥ دقائق)

### ١. أنشئي مستودع جديد
- ادخلي [github.com](https://github.com) → New repository
- اختاري اسم **معقّد وغير مألوف** للخصوصية (مثلاً: `bdgt-x7k2-prv` بدل `budget`)
- اجعليه Public (للأسف، Pages المجاني يحتاج Public)
- اضغطي Create

### ٢. ارفعي الملف
- اضغطي **Add file → Upload files**
- ارفعي ملف `index.html`
- اضغطي **Commit changes**

### ٣. فعّلي GitHub Pages
- في المستودع، ادخلي **Settings → Pages**
- تحت **Source**، اختاري **Deploy from a branch**
- اختاري `main` و `/ (root)`
- اضغطي **Save**
- بعد دقيقة-دقيقتين، رابطك جاهز: `https://USERNAME.github.io/REPO-NAME/`

### ٤. أضيفي طبقة حماية فوراً
- افتحي الرابط لأول مرة
- ادخلي **الإعدادات → رمز الحماية**
- اختاري رمز ٤ أرقام أو أكثر
- الآن حتى لو حد عرف الرابط، ما يقدر يدخل

### ٥. ضيفيها لشاشة آيفون
- افتحي الرابط في Safari
- اضغطي زر المشاركة → **إضافة إلى الشاشة الرئيسية**
- صارت أيقونة مستقلة على شاشتك

## المزامنة بين الأجهزة

**المشكلة:** البيانات محفوظة في متصفح كل جهاز لحاله.

**الحل اليدوي (موجود في الموقع):**
- في الجهاز الأول: الإعدادات → **تصدير** → احفظي ملف `.json` (مثلاً في iCloud Drive)
- في الجهاز الثاني: الإعدادات → **استيراد** → اختاري نفس الملف

سوّيها كل أسبوع كنسخة احتياطية أيضاً.

---

## ملاحظات مهمة

- **الخصوصية:** GitHub Pages المجاني = الرابط متاح علناً. اعتمدي على عدم وضوح الرابط + رمز الحماية. لو الخصوصية حرجة، انقلي الموقع لـ **Vercel** أو **Netlify** اللي يدعمان حماية بكلمة سر حقيقية على مستوى الخادم.
- **البيانات:** كل شي محفوظ محلياً في المتصفح (localStorage). لا يُرسل إلى أي خادم. صدّري نسخ احتياطية بانتظام لأن تنظيف بيانات المتصفح يمحي كل شي.
- **الرسائل:** الموقع لا يقرأ الرسائل تلقائياً (آبل لا تسمح بذلك). يحلّل ما تلصقينه فقط.

---

## ترقية محتملة لاحقاً

إذا كبر الاستخدام واحتجتِ مزامنة سحابية حقيقية، الخيارات:
- **Supabase** (مجاني، يحتاج إعداد قاعدة بيانات)
- **GitHub Gist API** (يحتاج Personal Access Token)
- **Firebase** (مجاني لاستخدامك)

كلها تتطلب إضافة كود للمزامنة، لكن إذا أردتِها لاحقاً، البنية الحالية جاهزة للتوسعة.
